== Description ==

**What is Takl.ink?** 

[تک لینک](https://takl.ink/) or Takl.ink is a tools to make a bio link with multiple links. You can use TakL.ink as Instagram bio link or other social networks like Telegram, Facebook, Twitter, ...

After register on [takl.ink](https://takl.ink/) you have a bio link like **takl.ink/username**. You can add multiple links, text, video, sound and etc. 

****

**What dose this plugin do for me?**

This plugin allow you to use onee page of your site domin as your takl.ink bio link Url.

**takl.ink/username => yoursite.com/username**

****

**Takl.ink WordPress plugin Features** 

* Use your custom domain instead of takl.ink domain for your bio link
* Select which page show bio link
* Easy to use

== Installation ==

To install and using Takl.ink please follow these steps:

1. Create an account on [Takl.ink](https://takl.ink/auth/)
2. log in to your takl.ink account and navigate to [Panel page](https://takl.ink/panel/) page.
3. Click on account link
3. Copy **Api Token** from opened section.
4. Install this wordpress plugin and active it.
5. After plugin installed, **Navigate to Dashboard > Settings > TakL.ink**
6. Paste Api token at **Api Token** field
7. Select page that you want to use as your bio page URL
8. Save changes and visit your selected page and Enjoy it!
